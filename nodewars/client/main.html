Bienvenue dans Nodewars. Le but de ce jeu est de créer une intelligence artificielle capable de survivre à des vagues successives d'ennemis. Pour cela tu disposes de plusieurs fonctions de base :
  • getNearest(int nb)
  • shoot(int x, int y)
  • move(int x, int y)
